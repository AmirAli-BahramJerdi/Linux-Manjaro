# تمرین‌های فصل ۴ – مدیریت پکیج‌ها

### تمرین ۱: آپدیت سیستم
```bash
sudo pacman -Syu
```

### تمرین ۲: نصب و حذف یک نرم‌افزار
```bash
sudo pacman -S htop
htop
sudo pacman -R htop
```

### تمرین ۳: جستجوی یک پکیج
```bash
sudo pacman -Ss python
```

### تمرین ۴: نمایش اطلاعات یک پکیج
```bash
sudo pacman -Qi python
```

### تمرین ۵: پاک کردن کش
```bash
sudo pacman -Sc
```

### تمرین ۶: نصب یک برنامه از AUR با yay
```bash
yay -S google-chrome
```
