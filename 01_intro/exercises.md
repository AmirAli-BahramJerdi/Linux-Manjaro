# تمرین‌های فصل ۱ – مقدمات لینوکس

### تمرین ۱: پیدا کردن مسیر فعلی
```bash
pwd
```

### تمرین ۲: لیست پوشه‌های Home
```bash
ls -la ~
```

### تمرین ۳: دیدن فایل یوزرها
```bash
cat /etc/passwd
```

### تمرین ۴: دیدن اطلاعات CPU
```bash
cat /proc/cpuinfo
```

### تمرین ۵: دیدن لاگ سیستم
```bash
ls /var/log
```
